# RV암기 사용 안내

요한계시록 암송을 위한 iOS 최적화 웹앱입니다. (반응형 — macOS Safari/Chrome에서도 사용 가능)

## 파일 구성
- `index.html` — 앱 전체 (디자인 + 기능)
- `bible-data.js` — 요한계시록 본문 데이터 (★ 직접 입력 필요)

## 1. 본문 채우기 (가장 먼저 할 일)

저작권 문제로 **개역한글** 성경 본문은 비워 두었습니다. `bible-data.js` 파일을 열어
아래 형식대로 채워주세요.

```js
1: [
  "예수 그리스도의 계시라 이는 하나님이 그에게 주사 ... 알게 하신 것이라",
  "요한은 하나님의 말씀과 예수 그리스도의 증거 ... 다 증거하였느니라",
  "이 예언의 말씀을 읽는 자와 듣는 자들과 ... 때가 가까움이라",
],
```

- 절 번호(1, 2, 3...)는 적지 않고 **본문 내용만** 큰따옴표 안에 넣습니다.
- 절마다 쉼표(`,`)로 구분된 별도의 문자열 한 줄입니다.
- **개역한글 원문 그대로** 띄어쓰기를 정확히 입력해야 백지시험 채점이 정확합니다
  (개역한글은 개역개정과 어휘·어미·띄어쓰기가 다른 부분이 있으니 혼용되지 않도록 주의해주세요).
- 장 전체를 비워두면(`[]`) 그 장은 범위 선택에서 자동으로 비활성화됩니다.
- 혹시 절 번호가 문장 맨 앞에 실수로 같이 들어가도(예: `"5 또 충성된..."`)
  앱이 자동으로 숫자를 제거하니 크게 신경 쓰지 않아도 됩니다.

다 채운 후 `index.html`을 더블클릭(또는 브라우저로 열기)하면 바로 적용됩니다.

## 2. 기능 요약

**성경보기**
요한계시록 전체를 장별로 넘기며 읽습니다.

**단어시험**
- 범위(장)와 난이도(쉬움 20% / 보통 40% / 어려움 65% 빈칸)를 선택합니다.
- 절마다 단어 일부가 빈칸으로 제시되며, 직접 입력해 채웁니다.
- **채점하기**: 맞은 단어는 파란 글씨, 틀린 단어는 빨간 테두리로 표시됩니다.
- **다시**: 같은 빈칸 구성으로 입력값만 초기화합니다.

**백지시험**
- 범위(장)를 선택합니다.
- 절마다 빈 입력칸이 제공되며, 문장 전체를 직접 입력합니다.
- **채점하기**: 단어/띄어쓰기까지 정확히 일치하면 파란 테두리,
  하나라도 다르면 빨간 테두리 + 아래에 정답이 표시되고 틀린 단어만 밑줄 표시됩니다.
- **다시**: 입력값을 모두 지웁니다.

## 3. iPhone 홈 화면에 추가하기 (선택)

Safari에서 `index.html`을 열고 공유 버튼 → "홈 화면에 추가"를 누르면
네이티브 앱처럼 아이콘으로 사용할 수 있습니다.

## 4. 맥북/아이맥에서 사용

`index.html`을 Safari나 Chrome으로 열면 됩니다. 480px 폭의 모바일형 카드 레이아웃이
화면 중앙에 그대로 보이며, 별도 수정 없이 동작합니다.
